Cisco IT Essentials
Virtual Desktop

Obtención segura

  1. Inscribirse en el curso IT Essentials dentro de Cisco Networking Academy (netacad.com), la plataforma oficial de Cisco.
  2. Ingresar con la cuenta del curso y ubicar la sección de recursos o actividades prácticas ("Virtual Desktop Activity").
  3. Descargar el archivo únicamente desde el enlace oficial que entrega la plataforma o el instructor.
  4. Descomprimir el archivo ZIP y abrir el index.html incluido para iniciar la actividad en el navegador.

⚠ Evitar descargarlo de sitios de terceros o enlaces "gratis": no garantizan la versión correcta ni la seguridad del archivo.

Requisitos del sistema

  • Windows 7/8/10/11 o macOS con navegador compatible.
  • Google Chrome (recomendado por Cisco para esta actividad).
  • Mínimo 2 GB de RAM y 200 MB de espacio libre.

Licenciamiento

Uso gratuito, pero restringido a estudiantes y docentes inscritos oficialmente en Cisco Networking Academy; no es un software de distribución pública independiente.

Logisim Evolution

Obtención segura

  1. Entrar al repositorio oficial en GitHub: github.com/logisim-evolution/logisim-evolution.
  2. Ir a la sección Releases y elegir el instalador según el sistema operativo (.msi Windows, .dmg macOS, .deb/.rpm/.snap Linux).
  3. Descargar solo desde ese repositorio oficial, evitando sitios espejo con software adicional no deseado.
  4. Ejecutar el instalador descargado y seguir el asistente paso a paso.

⚠ Los paquetes oficiales (.msi/.dmg) ya incluyen el entorno de Java; no es necesario instalarlo aparte salvo que uses el archivo .jar universal.

Requisitos del sistema

  • Java 21 o superior (incluido en los instaladores oficiales).
  • Compatible con Windows, macOS y Linux.
  • 4 GB de RAM recomendados y 100 MB de espacio en disco.

Licenciamiento

Software libre y de código abierto, bajo la Licencia Pública General de GNU (GPL‑3.0); gratuito para uso educativo, personal o institucional sin restricciones.
